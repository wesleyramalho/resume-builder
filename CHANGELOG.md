# Changelog

## 1.0.0

Initial stable release of MyPDFCV and its packages.

### Highlights

- Full resume builder with real-time PDF preview and export
- 7 professionally designed templates (modern, classic, minimal, executive, bold, balanced, clear)
- Multi-language support: English, Portuguese (BR), Spanish, Italian, Chinese, Japanese, German
- LinkedIn PDF import with automatic profile parsing
- In-browser AI resume suggestions powered by Transformers.js
- Drag-and-drop section reordering
- Photo upload with editor
- Dark mode support

### Packages

- **@mypdfcv/pdf-core** — Resume types, templates, validation schemas, and server-side PDF generation via `@react-pdf/renderer`
- **@mypdfcv/i18n** — Internationalization with 7 bundled locales and server/client entry points
- **@mypdfcv/mcp-server** — MCP server for AI-powered resume PDF generation (separate repo)
